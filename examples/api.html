toolbar.toolbarHandlers.insert(type:string)

type 效果
'hr' 删除线

'code' 代码块

'toc' 目录
'link' 超链接
'normal-table' 插入3行5列的表格
'normal-table-row*col' 如normal-table-2*4插入2行(包含表头是3行)4列的表格
